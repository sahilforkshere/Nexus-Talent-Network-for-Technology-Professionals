JOBS-SVC  (localhost:4003)
================================================================================

Post a job (needs token):
  mutation {
    postJob(input: {
      title: "Backend Engineer"
      company: "Google"
      location: "Bangalore"
      job_type: "FULL_TIME"
      experience_level: "MID"
      salary_min: 2000000
      salary_max: 3000000
      description: "Build scalable Go microservices for Google Search"
    }) {
      job_id title company
    }
  }

List all jobs (no token needed):
  query {
    listJobs { job_id title company location }
  }

Get single job:
  query {
    getJob(job_id: "<job_id>") { title company description }
  }

Search jobs (Elasticsearch):
  query {
    searchJobs(keyword: "Go") { title company location }
  }

Valid job_type values:    FULL_TIME | PART_TIME | CONTRACT | INTERNSHIP
Valid experience_level:   INTERN | JUNIOR | MID | SENIOR | LEAD

On postJob, three things happen automatically:
  1. Saved in PostgreSQL
  2. Indexed in Elasticsearch (searchable)
  3. Kafka publishes job_posted event


================================================================================
FEED-SVC  (localhost:4004)
================================================================================

Create a post (needs token):
  mutation {
    createPost(content: "Just joined Nexus! Excited to connect.") {
      post_id content created_at
    }
  }

Get your feed (needs token):
  query {
    getFeed {
      post_id user_id content created_at
    }
  }

How the feed works:
  createPost → saves to PostgreSQL posts table (author_id column)
             → pushes post_id to Redis sorted set  key: feed:{user_id}
  getFeed    → reads post IDs from Redis (newest first)
             → fetches full content from PostgreSQL
